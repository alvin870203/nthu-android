# 物件辨識Android藍芽控制自走車
微算機導論 | Introduction to Microcomputer | NTHU | 洪健中教授 | 2018 Fall

Android app透過手機鏡頭與YOLO神經網路進行物件偵測(object detection)，並透過藍芽模組控制自走車避障與追蹤。
